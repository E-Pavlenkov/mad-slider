Example 1

Simple responsive slider:

  • One slide: "slidesPerView":1
  • Default arrows: "arrows":true
  • Default counter: "counter":true
  • No loop: "loop": false

Example 2

Simple responsive slider:

  • One slide: "slidesPerView":1
  • Custom arrows: "arrows":true, "prev": "#prev", "next": "#next"
  • Loop: "loop": true
  • Counter: "counter": "#counter"
  • Pagination dots: "pagination": true
  • Auto height: "height": "auto"
